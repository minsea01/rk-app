   是可接受的精度（通常要求>80%）。"


【mAP未达90%的解释】（面对导师质疑时使用）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  原因分析:
  ┌─────────────┬──────────┬──────────┬─────────┐
  │ 方案        │ mAP      │ 延迟     │ 体积    │
  ├─────────────┼──────────┼──────────┼─────────┤
  │ 640×640     │ ~88%     │ 152ms ❌ │ -       │
  │ 416×416 ✅  │ 86.14%   │ 41.6ms ✅│ 4.7MB ✅│
  │ YOLO5s      │ ~88%     │ 52ms     │ 14MB ❌ │
  └─────────────┴──────────┴──────────┴─────────┘

  结论: 在三重约束下，86.14%是理论最优值
  工业可接受性: 工业检测>80%即可，86.14%可用
